# Questions Module

This module manages Questions, representing the third level in the Schema Builder (Edition -> Reform Area -> Action Point -> Question).

## Features
- **Deep Hierarchical Validation**: Enforces conditions all the way to the parent Edition.
- **Tenant Validation**: Prevents data access across organizations.
- **Auto-Sequencing**: Supports auto `displayOrder`.
- **Audit Logging**: Deep integration with `AuditService`.
